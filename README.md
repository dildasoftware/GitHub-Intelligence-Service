## 🛠️ Architecture

Built using **Clean Architecture** principles and **Domain-Driven Design (DDD)**.

*   **Backend:** .NET 9.0 Web API
*   **Frontend:** React + Vite + TypeScript
*   **Database:** SQL Server (EF Core Code First)
*   **Testing:** xUnit (Domain Logic Tests)
*   **Containerization:** Docker & Docker Compose

### Project Structure
```
📂 src
 ├── 📂 Api            # Controllers, Middleware (Global Exception Handling)
 ├── 📂 Application    # Business Logic, DTOs, Interfaces (CQRS-like)
 ├── 📂 Domain         # Enterprise Logic, Entities (Rich Model)
 └── 📂 Infrastructure # External Services (GitHub API), Database Context
```

## 🚀 Getting Started

### Prerequisites
*   .NET 9.0 SDK
*   Node.js (v18+)
*   Docker Desktop (Optional)

### Running Locally (Classic)

1.  **Backend:**
    ```bash
    cd src/GitHubIntelligenceService.Api
    dotnet run
    ```
2.  **Frontend:**
    ```bash
    cd client
    npm install
    npm run dev
    ```

### Running with Docker 🐳 (Recommended)

One command to rule them all:
```bash
docker-compose up --build
```
Access the app at `http://localhost:3000`.
